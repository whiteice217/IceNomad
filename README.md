## 🐧 About IceNomad

IceNomad is a native iOS client for the Reticulum Network Stack, designed to bring decentralized communications to mobile devices.

Whether connecting through TCP to a remote transport node or directly over LoRa using an RNode, IceNomad aims to make encrypted mesh networking simple, intuitive, and accessible from your pocket.

The long-term goal is to provide a complete mobile experience for the Reticulum ecosystem, including:

- Reticulum networking
- LXMF messaging
- NomadNet browsing
- Identity management
- Remote node connectivity
- LoRa mesh communications
