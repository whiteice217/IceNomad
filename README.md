# 🐧 IceNomad

<img src="https://raw.githubusercontent.com/whiteice217/IceNomad/main/Images/WanderingPengiun.png" width="250">

A penguin-powered Reticulum networking client for iOS.

IceNomad is an experimental mobile interface for exploring decentralized,
encrypted mesh communication using Reticulum, LXMF, LoRa, and remote nodes.

## Features

🐧 Reticulum networking  
📡 TCP and RNode connections  
❄️ LoRa mesh support  
🔐 Cryptographic identity management  
💬 LXMF messaging  
🌎 NomadNet exploration  
🚀 Real-time connection status
